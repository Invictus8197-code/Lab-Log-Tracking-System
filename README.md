# Lab Log Tracking System

## Project Status

Currently Under Development (Approximately 60% Complete)

## Overview

The Lab Log Tracking System is a web-based application designed to monitor and manage student activities inside a computer laboratory. The system helps faculty track lab usage, maintain records, and monitor student sessions efficiently.

## Features Implemented

- User Login Interface
- Student Dashboard
- Faculty Dashboard
- Firebase Integration
- Responsive User Interface
- Activity Tracking Structure

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Firebase
- Bootstrap
- EmailJS

## Planned Features

- Automatic Session Tracking
- Daily Report Generation
- Email Notifications
- Attendance Monitoring
- Database Record Management
- Admin Controls

## Project Structure

- index.html – Login Page
- student-dashboard.html – Student Panel
- faculty-dashboard.html – Faculty Panel
- style.css – Styling
- script.js – Client-side Functionality
- firebase.js – Firebase Configuration

## Future Improvements
